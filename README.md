## Task 1

1. Clone this repo into VS Code:
```
git clone https://github.com/thatcatfromspace/grokking-git
```
   
2. Create your own branch `dev/<roll-number>`
3. Create a `README.md` with a fun fact about anything
4. Commit and push to your branch!

## Task 2

1. Fork this repo to your account. Click the "Fork" button on the top-right of your screen - copying just the `master` branch is fine
2. Clone your fork into VS Code
3. Find your class' folder, create a file with your roll number as your name and fill it with more facts
4. Push the changes into your fork of the repo
5. Upon coming back to [this](https://github.com/thatcatfromspace/grokking-git/) page, head into the "Pull requests" tab on top and create a pull request with your modified fork
6. Approve your own pull request to see your file being integrated into the codebase!
